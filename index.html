Features

Everything you need in one binary

A single Go binary that connects your AI brain to the real world — no microservices, no containers, no complexity.

Telegram AI Chat

Conversational AI via Telegram with persistent history, photo analysis, document processing, and multi-user support with admin approval.

Voice Calls

Real-time voice AI powered by Gemini Live. Minerva makes and answers phone calls on your behalf — reservations, inquiries, anything.

Remote Code Agents

Connect Claude Code instances from any machine via WebSocket. Dispatch coding tasks — git, builds, debugging — and get results in Telegram.

Email Integration

Send and receive emails via Resend. The AI drafts and sends messages on your behalf, and incoming emails are routed through webhooks.

Persistent Memory

Minerva remembers facts about you across conversations — preferences, ongoing projects, important details. Fully AI-managed with a lightweight SQLite backend.

Scheduled Tasks

Schedule one-off or recurring tasks on remote agents — daily reports, weekly builds, automated maintenance. Managed via CLI or the AI brain.

Architecture

Simple by design

One binary. One database. No orchestrators, no message queues, no YAML files. Minerva connects directly to the services it needs.

Quick Start

Up and running in 60 seconds

All you need is Go 1.24+, Claude CLI installed, and a Telegram bot token.

terminal
# Clone the repository git clone https://github.com/kidandcat/minerva.git cd minerva # Configure (set TELEGRAM_BOT_TOKEN and ADMIN_ID) cp .env.example .env nano .env # Build and run go build -o minerva . ./minerva

Docker

Also available: docker compose up -d with the included docker-compose.yml

VPS Deploy

Cross-compile with GOOS=linux GOARCH=amd64 go build and deploy with systemd

Caddy / Nginx

Put behind a reverse proxy for HTTPS — needed for voice calls and email webhooks

Agent Setup

Build the agent binary separately: cd agent && go build, then connect via WebSocket

AI Tools

Built-in capabilities

The AI brain can invoke these tools autonomously during conversations to get things done on your behalf.

update_memory Persist important facts about you across all conversations
send_email Send emails via Resend from verified domains
make_call Initiate voice calls via Twilio with real-time Gemini AI
run_claude Dispatch coding tasks to remote Claude Code agents
list_claude_projects List projects available on connected agents
create_task Launch long-running background tasks with progress tracking
get_task_progress Check status and progress of running background tasks
run_code Execute JavaScript in a sandboxed environment (Goja)
Telegram

Bot commands

Quick actions available directly in your Telegram chat.

/start Show welcome message and available commands
/clear Clear conversation context and start fresh
/token Update Claude OAuth token (admin only)
